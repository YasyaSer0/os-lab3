## Завдання для попередньої підготовки

### Glossary of Basic English Terms

**1. Command - An instruction entered in the terminal to perform a specific action in the operating system.**

**2. Shell - A program that interprets user commands and passes them to the operating system for execution.**

**3. Command Line Interface (CLI) - A text-based interface that allows users to interact with the operating system by typing commands.**

**4. Prompt - A text displayed in the terminal indicating that the system is ready to receive a command.**

**5. Argument - Additional information provided to a command (for example, a file or directory name).**

**6. Option (Parameter) - A modifier that changes the behavior of a command. It usually begins with '-' or '--'.**

**7. Flag - A short option that enables or disables a specific function of a command.**

**8. Directory - A folder used to organize and store files and other directories.**

**9. File - A data storage object in the file system.**

**10. Path - The location (address) of a file or directory in the file system.**

**11. Environment Variable - A variable that stores system or user configuration values.**

**12. Alias - A custom shortcut name for a command.**

**13. History - A list of previously executed commands.**

**14. Man Page (Manual Page) - A built-in documentation page for a command, accessed using the 'man' command.**

**15. Export - A command used to make a variable available to child processes.**

**16. Unset - A command used to remove a variable.**

**17. Echo - A command used to display text or variable values in the terminal.**

**18. Execution - The process of running a command or program.**
